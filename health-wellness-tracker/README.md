# Health and Wellness Tracker

## Project Setup

### Backend

1. Install Scala and sbt.
2. Navigate to the `backend` folder.
3. Run `sbt run`.

### Frontend

1. Install Node.js and npm.
2. Navigate to the `frontend` folder.
3. Run `npm install`.
4. Run `npm start`.

## Features

- User Authentication
- Health Metrics Tracking
- Workout Planner
- Dietary Log
- Health Insights
